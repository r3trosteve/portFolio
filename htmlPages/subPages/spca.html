This was for a class project in Yoobee where we were learning Wordpress, we were to pick a site they we like, I'm an animal lover and naturally chose the SPCA, and make site that could blend be an add on. I use wordpress plugins and all content and imagery are from the SPCA's actual website, I try to use the colours, styles and fonts that they use and focus on a few pages. I made a shop page and donate page using by creating custom templates as well as a slideshow, it's mostly a very customisable website that could suit many companies.
