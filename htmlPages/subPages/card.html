This was a quick test project for Yoobee where students had to make a webpage where users could order a card that tehy would basically personalise, I made 3 covers (congratulations, sorry and happy birthday) then let the user input text, which they could style in font and sizing, then decorate the other side of the page with four options, pick a colour for the decoration as well as a background colour, there are also validation errors. Website is also responsive.
